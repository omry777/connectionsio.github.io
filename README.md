# 🎮 Connections - משחק הקשרים בעברית

משחק Connections בעברית בהשראת משחק ה-NYT Connections, עם מערכת ניהול חידות אוטומטית וסטטיסטיקות מתקדמות.

## ✨ תכונות

### 🎯 תכונות משחק
- **חידות יומיות** - חידה חדשה כל יום
- **4 קבוצות** - מצא 4 קבוצות של 4 מילים עם קשר משותף
- **4 נקודות חיים** - יש לך 4 הזדמנויות לטעות
- **אנימציות יפות** - חוויית משתמש מעוצבת ואטרקטיבית

### 📊 סטטיסטיקות ואנליטיקה
- **סטטיסטיקות אישיות**:
  - אחוז הצלחה
  - רצף ניצחונות (streak)
  - ממוצע טעויות
  - משחקים מושלמים (ללא טעויות)
  - התפלגות תוצאות

- **סטטיסטיקות גלובליות**:
  - סך משחקים היום
  - אחוז הצלחה כללי
  - ממוצע טעויות גלובלי

- **שיתוף תוצאות** - שתף את הישגיך עם חברים

### 🤖 יצירת חידות אוטומטית
- **מחולל חידות מובנה** - מערכת ליצירת חידות עם מאגר עשיר של קבוצות
- **חידות מגוונות** - קבוצות בנושאים שונים:
  - צבעים
  - בעלי חיים
  - כלי נגינה
  - עונות השנה
  - ועוד הרבה...

### 🎛️ לוח בקרה למנהלים
- צפייה בכל החידות במערכת
- יצירת חידות חדשות
- צפייה בהצעות משתמשים
- סטטיסטיקות מערכת

## 🚀 התקנה והפעלה

### דרישות
- דפדפן מודרני (Chrome, Firefox, Safari, Edge)
- שרת HTTP (או GitHub Pages)
- (אופציונלי) חשבון Firebase לסטטיסטיקות גלובליות

### הגדרת הפרויקט

1. **שכפל את הקבצים:**
```bash
git clone [your-repo-url]
cd connectionsio.github.io
```

2. **הגדר Firebase (אופציונלי):**
   - צור פרויקט ב-[Firebase Console](https://console.firebase.google.com/)
   - העתק את פרטי ההגדרה ל-`firebase_config.js`:
   ```javascript
   const firebaseConfig = {
     apiKey: "YOUR_API_KEY",
     authDomain: "YOUR_AUTH_DOMAIN",
     projectId: "YOUR_PROJECT_ID",
     storageBucket: "YOUR_STORAGE_BUCKET",
     messagingSenderId: "YOUR_MESSAGING_SENDER_ID",
     appId: "YOUR_APP_ID"
   };
   ```

3. **הפעל שרת מקומי:**
```bash
# Python 3
python -m http.server 8000

# או Node.js
npx http-server
```

4. **פתח בדפדפן:**
   - משחק: `http://localhost:8000`
   - לוח בקרה: `http://localhost:8000/admin.html`

## 🎲 איך להוסיף חידות חדשות

### דרך 1: באמצעות לוח הבקרה (מומלץ)
1. פתח את `admin.html`
2. לחץ על "צור חידה ליום הבא"
3. בדוק את החידה שנוצרה
4. לחץ על "שמור חידה" להורדת קובץ JSON מעודכן
5. העלה את הקובץ לשרת

## 🔧 התאמה אישית

### שינוי צבעי קבוצות

```javascript
const colors = ['#f44336', '#4caf50', '#9c27b0', '#2196f3'];
```

```javascript
  // ...

```

### שינוי עיצוב

## 🐛 פתרון בעיות נפוצות

### המשחק לא טוען
- ודא שאתה מריץ דרך שרת HTTP (לא file://)
- בדוק את ה-Console בדפדפן לשגיאות
- ודא שקובץ `puzzles.json` תקין

### הסטטיסטיקות לא נשמרות
- הסטטיסטיקות המקומיות נשמרות ב-localStorage
- נקה את ה-cache של הדפדפן אם יש בעיה

### Firebase לא עובד
- ודא שמילאת את הפרטים ב-`firebase_config.js`
- הסטטיסטיקות המקומיות יעבדו גם ללא Firebase
